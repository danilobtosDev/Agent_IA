# Passos do GIT:
Passo-a-passo do Git após ter criado o novo arquivo:

# Quando quizer ver o Log de envio de arquivos para o Git.
git log

# Adicionar arquivo para ser enviado para Git:
git add .

# Comitar arquivo no repositório. "Texto informativo".
git commit -m ""

# Subir e salvar arquivo no Git.
git push -u origin main
